# SERAVOLVE 2.0 SEARCH DISCOVERABILITY SYSTEM

## Objective
Make Seravolve discoverable through Google Search, Google Shopping, ChatGPT, Gemini, Claude, Perplexity, Voice Search, and AI Overviews. Optimize for every way people discover brands.

---

# 1. Site Architecture
```
Home
├── Find My Routine
├── Shop
├── Routines
├── Learn
├── About
└── Contact
```
*Never exceed 3 clicks to reach any page.*

---

# 2. URL Rules

**Products & Routines (Shopify Native)**
Due to Shopify's URL constraints, use native URL structures but keep slugs clean:
- `/products/clarifying-face-wash`
- `/products/radiance-toner`
- `/products/daily-essentials-routine`
- `/collections/routines`

**Learn / Blog**
- `/blogs/learn/how-to-build-a-skincare-routine`
- `/blogs/learn/how-to-layer-skincare`

---

# 3. Page SEO Template
Every page follows this format:
- **H1:** One only.
- **Intro:** 40–80 words. Immediately answer: *What is this page?*
- **Body:** Organized using H2s. No skipping heading levels.
- **FAQ:** Minimum 5 questions.
- **CTA:** One primary CTA.

---

# 4. Metadata Rules

**Product Title Formula:**
`Product Name | Concern | Seravolve`
*(Example: Barrier Repair Moisturizer | Seravolve)*

**Meta Description Formula (140–155 chars):**
`What it is. Who it's for. Free Shipping. Brand.`
*(Example: A lightweight moisturizer designed to support your skin barrier with ceramides, peptides and squalane. Free shipping across India. Seravolve.)*

---

# 5. Image SEO
- **Filename:** `clarifying-face-wash.webp` (Never `IMG_00452.webp`)
- **Alt Text Formula:** `Product` `Purpose` `Brand`
*(Example: Seravolve Clarifying Face Wash for daily cleansing)*

---

# 6. Internal Linking Rules
- Homepage -> Routine
- Routine -> Products
- Products -> Routine
- Blog -> Routine
- FAQ -> Routine
*No orphan pages.*

---

# 7. Structured Data
**Implement Now:**
- Organization, Website, Breadcrumb, Product, FAQ, Collection
**Implement Later:**
- Article, HowTo, VideoObject, Review

---

# 8. LLM Optimization
Every page must answer these questions in plain language (Don't hide in accordions):
1. What is it?
2. Who is it for?
3. What concern does it help with?
4. How do I use it?
5. When should I use it?
6. What should I use next?

---

# 9. AI Citation Rules
Write so AI can quote your content. Clear and Factual.
- **Bad:** Our revolutionary proprietary formula...
- **Good:** The Barrier Repair Moisturizer is designed for people with dry or weakened skin barriers and is used after serum.

---

# 10. Entity Strategy
**Own these primary concepts:** Simple skincare, Skincare routines, Beginner skincare, Daily skincare routine.
**Own these secondary concepts:** Barrier repair, Glow routine, Daily sunscreen, Daily cleansing.
*Don't chase ingredient keywords. Own routine keywords.*

---

# 11. Content Clusters
Instead of random blogs, create content clusters based on topics (e.g., Skincare Routine -> Morning Routine -> Night Routine -> Layering Products -> Routine Order). 

---

# 12. FAQ Database
One reusable FAQ library. Categories: Products, Routines, Shipping, Returns, General. Displayed dynamically across pages.

---

# 13. Blog Strategy
One article every week.
**Format:** Problem -> Why it happens -> Routine -> Products -> FAQs
**Never:** Problem -> Ingredients -> Affiliate links. Stay on-brand.

---

# 14. Technical SEO
- XML sitemap, Robots.txt, Canonical URLs
- WebP images, Lazy loading, Core Web Vitals
- Mobile-first, HTTPS only

---

# 15. Performance Targets
- Performance: 95+
- Accessibility: 95+
- Best Practices: 100
- SEO: 100

---

# 16. Claude Rules
**Never:**
- Keyword stuff, Duplicate H1s, Duplicate meta descriptions, Auto-generate AI-sounding blog content.
**Always:**
- Write naturally, Link related routines, Use semantic headings, Answer user intent first.

---

# 17. Future GEO (Generative Engine Optimization)
Every page should include a small factual summary.
*Example: The Daily Glow Routine is a four-step skincare routine for people with dull-looking skin. It includes a cleanser, toner, vitamin C serum and sunscreen for daily morning use.*
